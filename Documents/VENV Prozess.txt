:: .env erstellen:
python -m venv venv

:: venv aktivieren (Windows - CMD):
venv\Scripts\activate

:: venv aktivieren (Windows - Bash):
source venv/Scripts/activate

:: venv aktivieren (Linux - Bash):
source venv/bin/activate

:: venv aktivieren (Linux - Fish):
source venv/bin/activate.fish

:: Change Directory to src to run Main.py
cd .\src

::Start Programm
python Main.py

:: venv deactivieren (Windows):
venv\Scripts\deactivate

:: venv deaktivieren (Linux - Bash):
source venv/bin/deactivate

:: venv löschen (Windows):
rd /s /q venv

:: venv löschen (Linux):
rm -rf venv/

:: Libs installieren:
pip install --trusted-host pypi.org --trusted-host files.pythonhosted.org --verbose python-dotenv Setuptools pyinstaller

:: Lib deinstallieren
pip uninstall LIB_NAME

:: Requirements erstellen:
pip freeze > requirements.txt

:: Requirements installieren:
pip install --trusted-host pypi.org --trusted-host files.pythonhosted.org --verbose -r requirements.txt

:: show outdated:
pip list --outdated

:: update all (PowerShell):
pip list --outdated --format=json | ConvertFrom-Json | ForEach-Object { $_.name } | ForEach-Object { pip install --trusted-host pypi.org --trusted-host files.pythonhosted.org --upgrade $_ }

::Update Pip
python.exe -m pip install --upgrade pip

::Check Python Version
python --version
::Check Python locations
where python

::Look up current Version online
winget search python

::Update Python
winget install Python.Python.3 --version &lt;3.12.4&gt;

:: Creating a executable with pyinstaller:
pip freeze > requirements.txt
pip install --trusted-host pypi.org --trusted-host files.pythonhosted.org pyinstaller
:: Standard
pyinstaller Main.py

::Options
pyinstaller --onefile Main.py 
or
pyinstaller --onedir Main.py

::Folders (Template, Out and Data), .env and config.ini file have to be added to Main.exe folder.

:: Creating a executable with cx_Freeze:
- add Main.py to build.py
- add folders to build.py
- add config.ini to build.py
- add requirements to build.py
